# SecureAaegix - Cybersecurity Training Platform

A comprehensive cybersecurity training and certification platform offering industry-recognized courses, hands-on labs, and professional security services.

## 🚀 Features

### 👤 User Features
- User registration & login with email verification
- OTP-based email authentication
- Password reset functionality
- User profile management
- Dark/Light mode toggle
- Responsive design for all devices

### 📚 Courses
- Industry-recognized certifications (OSCP, OSWE, CRTO, eJPT, etc.)
- Course enrollment system
- Progress tracking
- Certificate generation upon completion

### 🛡️ Services
- Penetration Testing
- Vulnerability Assessment
- Incident Response
- Red Team Operations

### 📝 Blog
- Cybersecurity articles and tutorials
- Search and filter functionality
- Comment system
- Category-based browsing

### 👑 Admin Features
- User management
- Course management
- Blog post approval
- Contact form management
- Certificate issuance

## 🛠️ Tech Stack

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM for MongoDB
- **Passport.js** - Authentication
- **Brevo** - Email service

### Frontend
- **EJS** - Template engine
- **Tailwind CSS** - Styling
- **JavaScript** - Client-side interactivity
- **Font Awesome** - Icons

### Development Tools
- **Nodemon** - Auto-restart during development
- **dotenv** - Environment variables
- **bcryptjs** - Password hashing
